PlantLink




¡Bienvenido a PlantLink!

PlantLink fue elaborado de manera independiente con el objetivo de ser una empresa dedicada al control de plagas, a través de técnicas innovadoras de alta calidad, tecnología y costos competitivos. Con nuestra solución podrás disfrutar de un ambiente saludable para tus plantas y una forma fácil y efectiva de monitorear su salud y crecimiento.




Mantenimiento




Mantenimiento preventivo de software

Nuestro mantenimiento preventivo de software nos permite mirar hacia el futuro y hacer los cambios necesarios para garantizar que nuestro software funcione como se desea durante el mayor tiempo posible. Además, nuestro mantenimiento perfectivo de software tiene como objetivo ajustar el software agregando nuevas características según sea necesario y eliminando características que son irrelevantes o no efectivas en el software dado, lo que mantiene el software relevante a medida que el mercado y las necesidades del usuario cambian.


Mantenimiento preventivo del dispositivo

Realizamos mantenimiento continuo en nuestro hardware para asegurarnos de que siempre esté actualizado y funcione de manera óptima. Nuestro objetivo es ofrecer los mejores precios a nuestros clientes, mejorando la calidad del hardware mediante la implementación de mejoras continuas.





Seguridad




Seguridad del software

Para garantizar la protección de nuestros sistemas informáticos, implementamos contraseñas seguras y confiables que cumplen con los requisitos de nuestra organización. Esto abarca todas las contraseñas utilizadas en el sistema, incluyendo cuentas de usuario y servicios de red.


Seguridad del dispositivo

Nuestro dispositivo utiliza el protocolo UART y MQTT para transmitir paquetes de datos. Esto garantiza una alta encriptación de los datos, así como una comunicación peer-to-peer segura en lenguaje binario. Además, al no requerir cuentas de usuario para su uso, nuestro dispositivo reduce la exposición de la información del usuario a un ID único por dispositivo.





Requisitos




Requisitos del software

Nos enorgullece ofrecerte una experiencia de monitoreo de plantas de la manera más fácil y accesible posible. Nuestra página web es compatible con cualquier navegador, lo que significa que no importa si usas Google Chrome, Firefox, Safari o cualquier otro, podrás acceder y visualizar el monitoreo de tus plantas sin problemas.


Además, nuestra aplicación móvil es compatible con el sistema operativo Android, lo que te permite monitorear tus plantas en cualquier momento y en cualquier lugar desde la comodidad de tu dispositivo móvil. Todo lo que necesitas es una conexión estable a internet para que puedas estar siempre informado sobre la calidad del aire y tierra.


Requisitos del dispositivo

A pesar de que nuestro dispositivo opera con la tecnología LoRaWAN, su Gateway requiere un punto de acceso a Internet para enviar de forma segura los datos a nuestra base de datos. Por lo tanto, es necesario ingresar la red y la contraseña de una señal estable. Además, nuestros dispositivos son muy intuitivos y solo requieren el suministro de energía para funcionar activamente.

La garantía se ofrece para reemplazar inmediatamente el dispositivo en caso de cualquier defecto eléctrico o de diseño, para asegurar que los clientes reciban el nivel de calidad de servicio que merecen.
